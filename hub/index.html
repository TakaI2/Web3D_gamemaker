Web3D GameMaker

エディタ・ゲーム一覧（開発用ランチャー）

ゲーム

Swing Catch GAME
空中のオブジェクト/VRM NPC をキャッチ・振り回し・吹っ飛ばす。NPCはステート/表情/ラグドール。
FPS VRM NPC + Cloth GAME
FPS視点でマント付きVRM NPCと布シミュ。発射→ラグドール。
FPS + Cloth GAME
FPS移動＋GPU布シミュレーション。
Cloth Sim GAME
WebGPU 布物理シミュレーション（形状パラメータ・エクスポート）。

エディタ

Character Editor EDITOR
VRM NPC のステート（通常/警戒/攻撃/ダウン/復帰）に表情・視線・行動を設定し .npc.json として書き出す。
Cloth Editor EDITOR
VRMにマント等の布を着せ、ピン/コライダー/手グリップを設定して書き出す。
VRM / MMD / FBX Editor EDITOR
モデル読込・アニメ再生・リターゲット・FBX→VRMA 変換などの本体エディタ。
各リンクは開発サーバーの相対パス（本番デプロイでは個別配置）。
