Web3D GameMaker

エディタ・ゲーム一覧（開発用ランチャー）

ゲーム

TPS Flight GAME
サイキッカー(Joy_reborn)をTPS視点で操作する空中アクション。球面カメラ＋スプリング追従。浮遊物を目前に引き寄せ・射出。
Swing Catch GAME
空中のオブジェクト/VRM NPC をキャッチ・振り回し・吹っ飛ばす。NPCはステート/表情/ラグドール。
FPS VRM NPC + Cloth GAME
FPS視点でマント付きVRM NPCと布シミュ。発射→ラグドール。
FPS + Cloth GAME
FPS移動＋GPU布シミュレーション。
Cloth Sim GAME
WebGPU 布物理シミュレーション（形状パラメータ・エクスポート）。
Story Player GAME
3Dキャラ(npc.json)がしゃべり・動く線形ストーリーを再生。
Flow Player GAME
ストーリー→戦闘→勝敗で分岐するゲームフローを連続再生。

エディタ

Character Editor EDITOR
VRM NPC のステート（通常/警戒/攻撃/ダウン/復帰）に表情・視線・行動を設定し .npc.json として書き出す。
Cloth Editor EDITOR
VRMにマント等の布を着せ、ピン/コライダー/手グリップを設定して書き出す。
FX Editor EDITOR
地面＋壁の小部屋でNPCのtimeline(VRMA)を再生し、任意フレームにエフェクト(炎/煙/火花/竜巻/自作)を配置。発生位置をギズモで移動/回転、bloom、timeline.json に埋め込み保存。
FX Builder EDITOR
基本図形＋テクスチャ＋色/加算合成/UVスクロール/ねじれ/自転/脈動 のレイヤーを重ねて発光VFXのプリセットを作成。粒子レイヤーも混在可。public/fx に保存し FX Editor で使える。
Model Editor EDITOR
public/models/ の GLB をサムネ一覧から選び、ゲームに登場させる飛行オブジェクトを選別する。
Stage Editor EDITOR
俯瞰ビューでモデルをマウス配置。位置・回転を調整して swing-catch のステージ置物として保存する。
Ragdoll Editor EDITOR
NPCを崩して関節の回転制限を調整。グラブ点固定・関節/骨の可視化・即シミュ。ragdoll.json として保存。
Story Editor EDITOR
3Dストーリーのコマンド列を編集。アクター(npc.json)のセリフ・移動・モーション・カメラを設定し再生確認・保存。
Flow Editor EDITOR
ストーリー/戦闘ノードを接続し、勝敗で分岐するゲームフローを編集・保存。
VRM / MMD / FBX Editor EDITOR
モデル読込・アニメ再生・リターゲット・FBX→VRMA 変換などの本体エディタ。
各リンクは開発サーバーの相対パス（本番デプロイでは個別配置）。
