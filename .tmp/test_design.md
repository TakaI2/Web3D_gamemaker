# テスト設計書 - Cloth Preview

## 1. テスト方針

スタンドアロン HTML/JS のため自動テストは困難。
ブラウザ上での**手動テスト**を主体とし、再現手順を明確に記述する。
クリティカルなロジック（補間・JSON 入出力・イベント発火）は console.assert で検証する。

## 4. ロジック単体検証（ブラウザ console で実行）

```js
// 補間テスト
const kfMap = new Map([[0, 0.0], [30, 1.0], [60, 0.0]]);
console.assert(interpolateBlendShape(kfMap, 15) === 0.5,  'f=15 → 0.5');
console.assert(interpolateBlendShape(kfMap, 45) === 0.5,  'f=45 → 0.5');
console.assert(interpolateBlendShape(kfMap,  0) === 0.0,  'f=0 → 0.0');
console.assert(interpolateBlendShape(kfMap, 60) === 0.0,  'f=60 → 0.0');
console.assert(interpolateBlendShape(kfMap, 70) === 0.0,  'f=70 → last clamp');

// JSON ラウンドトリップテスト
timeline.grip.gripLeft.add(5); timeline.grip.gripLeft.add(20);
timeline.blendShape.set('happy', new Map([[0, 0.0], [15, 1.0]]));
const exported = exportTimeline();
importTimeline(exported);
console.assert(timeline.grip.gripLeft.has(5),               'gripLeft f=5 復元');
console.assert(timeline.grip.gripLeft.has(20),              'gripLeft f=20 復元');
console.assert(timeline.blendShape.get('happy')?.get(15) === 1.0, 'happy f=15 復元');
```
