新規地形

地形ブラシ（左ドラッグ）

ペイント

道路（スプライン）

クリックで点を追加（続けて引ける）／点をドラッグで移動（他の道路の点に吸着＝交差点）／右クリックで引き終わり。高さは常に地形に追従。

建物（差分編集）

箱クリック=選択→ドラッグ移動 / R回転 / Del削除。＋ボタン→地形クリックで設置。自動配置への差分だけ保存されるので、道路確定後に調整推奨。

水面

＋水面→地形クリックで設置（水位=その場の高さ+1m）。クリック選択→ドラッグ移動、スライダでサイズ/水位。

町プレビュー

自作道路があればそれを、無ければOSM道路を使用。建物はゲームと同じ配置ロジック（箱表示）。ブラシや道路編集に追従します。

保存 / 読込
